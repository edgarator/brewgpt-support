# BrewGPT Support

Support website for the BrewGPT iOS app.

## Live Site

https://yourusername.github.io/brewgpt-support

## Setup

1. Push this repo to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch" → main → / (root)
4. Your support site will be live at `https://yourusername.github.io/brewgpt-support`
